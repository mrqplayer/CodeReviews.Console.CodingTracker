This is Coding Tracker.
Its meant to be used to track time of coding, by simply inserting start and end time.
-------------------------------------------------------------------------------------


What i used to make it work:
---------------------------

c# language

Dapper

SqLite

Spectre console

Ai


Features
--------

Coding time is automatically calculated after entering start and end time.

If you want to use present date it can be easily inserted by simply clicking "t".

Program has validation of correct date format.

Thoughts
--------

It was my first time using dapper and spectre, so it took some time to write this code. Sqlite doesnt support date type, so dates are stored as strings.
Code has separate classes to keep it clean. Main challenge was to separate classes and make them work together properly. I hope that i will remember
what i did in this project so next projects it will be easier for me to make something similar work.
